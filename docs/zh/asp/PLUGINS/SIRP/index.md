## 插件配置

- 将配置文件 agentic-soc-platform/PLUGINS/SIRP/CONFIG.example.py 重命名为 CONFIG.py 使配置生效
- SIRP_URL为 SIRP 平台地址,如 http://192.168.241.128:8880 (私有部署) 或 https://www.nocoly.com (云服务)
- AppKey 对应 SIRP_APPKEY ,Sign 对应 SIRP_SIGN
