# Mem0 Plugin

## Before you start

- The Mem0 plugin depends on the `Embeddings`, `Huggingface`, `LLM`, `Neo4j`, and `Qdrant` plugins. Please ensure that these plugins are correctly configured and running.

## Configuration Method

- Rename PLUGINS/Mem0/CONFIG.example.py to CONFIG.py
- Fill in the configuration items according to the code comments
