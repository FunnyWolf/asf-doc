# AlienVaultOTX Plugin

## Configuration

- After registering an account, visit [AlienVault](https://otx.alienvault.com/settings) to obtain an API Key
  ![img.png](img.png)

- Rename the configuration file PLUGINS/AlienVaultOTX/CONFIG.example.py to CONFIG.py to activate the configuration

- Fill in the configuration items according to the code comments
