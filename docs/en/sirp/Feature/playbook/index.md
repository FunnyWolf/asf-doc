# Playbook

Automated playbook records associated with Alert/Case/Artifact.

## Kanban / View

![img.png](img.png)

## Detail

- Status

Current status of the Playbook, divided into four states: `Pending`, `Running`, `Success`, `Failed`.

- Type

Three types: `Case`, `Alert`, `Artifact`.

- Name

Name of the executed Playbook.

- Worksheet

Associated Worksheet.

- Job ID

Unique Job ID generated by the ASP backend.

- Row ID

Associated Worksheet rowId.

- Remark

Remark information for task execution.
