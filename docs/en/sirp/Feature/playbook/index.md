# Playbook

与 Alert/Case/Artifact 相关联的自动化剧本记录.

## Kanban / View

![img.png](img.png)

## Detail

- Status

Playbook 当前状态,分为 `Pendding` `Running` `Success` `Failed` 四种状态.

- Type

`Case` `Alert` `Artifact` 三种类型.

- Name

执行的 Playbook 名称.

- Worksheet

关联的 Worksheet.

- Job ID

ASF 后台生成的唯一 Job 编号.

- Row ID

关联的 Worksheet rowId

- Remark

任务执行的备注信息.
